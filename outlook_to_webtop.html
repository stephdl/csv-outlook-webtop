Outlook Contacts → WebTop

Converts your Outlook CSV export to a WebTop-ready CSV with automatic field mapping.

Click to select the Outlook CSV file
File → Open & Export → Import/Export → CSV (Contacts)

How to import in WebTop

  1. Right-click your address book → Import contacts
  2. Select the _webtop.csv file
  3. Step 2 — Delimiter: Comma, Text qualifier: ", Encoding: UTF-8
  4. Step 4 — all fields mapped automatically → Next → Start

How it works

Detects UTF-8 BOM or Windows-1252 automatically. Runs entirely in your browser — no data is uploaded.
